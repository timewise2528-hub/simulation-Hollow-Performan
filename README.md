# Hollow Performance: A Closed‑Loop Learning Theory of LLM Dependence

This repository implements the mathematical models and synthetic simulations from the paper  
*"Hollow Performance: A Closed‑Loop Learning Theory of LLM Dependence"*.

## Overview

The core idea: AI can increase **observed performance** while weakening **human mastery** – a state called *hollow performance*.

Two levels of formalisation are implemented:

1. **Simple risk model**  
   - Unverified delegation: `r = D(1-V)`  
   - Safe boundary: `D(1-V) < 1/(1+κ)`

2. **First‑principles dynamic model**  
   - Observed performance: `Y_t = 1 - (1-K_t)(1-A_t)`  
   - Durable learning:  
     `L_t = η(1-K_t) Q_t G(P_t,F_t,J_t,B_t) exp(-λτ/(1+ρS_t))`  
   - Mastery update: `K_{t+1} = K_t + L_t - δ_offload - δ_forget - δ_social`

All equations are implemented in [`src/hollow_performance.py`](src/hollow_performance.py).

| File / Directory | Description |
